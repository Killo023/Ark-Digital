# Project Images

This directory contains images for all projects showcased on the website.

## Image Structure

Each project folder should contain:
- `hero.jpg` - Main hero image for the project (1920x1080px recommended)
- `gallery-1.jpg`, `gallery-2.jpg`, `gallery-3.jpg` - Gallery images (1920x1080px recommended)

## Projects

1. **chernelang-physio/** - Chernelang Physiotherapy website
2. **fitguide-pro/** - FitGuide Pro fitness platform
3. **skainet-cleaning/** - Skainet Cleaning Services website
4. **dma-law/** - DMA Law firm website
5. **skills-to-furnish/** - Skills to Furnish International training institute

## Adding Images

1. Add your project screenshots/images to the respective project folder
2. Name them according to the structure above (hero.jpg, gallery-1.jpg, etc.)
3. The website will automatically use these images

## Image Optimization

- Use JPG for photographs, PNG for graphics with transparency
- Recommended max file size: 500KB per image
- Next.js will automatically optimize images on build
